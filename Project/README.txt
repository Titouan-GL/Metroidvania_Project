A MetroidVania Game made by Kevin Espiguinha, Clément Chevalier and Titouan Guibert-Lodé.
You play as a knight trying to go through the few levels the game presents you by fighting ennemies and avoiding spikes. In the end you will find a boss that you will try to defeat.

Move with the arrow keys
Jump with spacebar
attack with D

It's made on Linux and only executable on Linux. The codeblocks project are still in the file but not really usefull if you don't use codeblocks.
